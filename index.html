CSS Layout: Position Property
Position Properties Description Example
Static Default postion. Not affected by top, bottom, left, right properties. Positioned to normal flow of page #sticky { position: static; border: 1px, solid, blue }
Relative Element is positioned relative to its normal position. Position adjustments of top, right, bottom and left properties cause item to be adjusted away from normal position. Other content will not be affected to fill any gap left behind. div.relative {position: relative; right 30px, border: 3px solid black}
Fixed Positioned relative to the viewport. Position stays in same place even if screen is scrolled. Can be adjusted by top, right, bottom and left properties. Does not leave a gap in the page when moved. div.fixed { position: fixed; bottom: 10; right: 50; width: 200px; border: 3px solid gray; }
Absolute Positioned relative to the nearest ancestor, not the viewport as fixed is. If no ancestors, it uses the document body and moves with page scrolling. Elements are removed from the noraml flow and can overlap other elements. div.relative { position: relative; width: 400px; height: 200px; border: 2px solid blue;
Sticky Elements positioned based on user's scroll position. Toggles between relative and fixed depending on scroll position. Is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed. div.sticky { position: -webkit-sticky; /* Safari */ position: sticky; top: 0; background-color: green; border: 2px solid gray; }

All CSS Positioning Properties
Position Properties Description
bottom Sets the bottom margin edge for a positioned box
clip Clips an absolutely positioned element
left Sets the left margin edge for a positioned box
position Specifies the type of positioning for an element
right Sets the right margin edge for a positioned box
top Sets the top margin edge for a positioned box
